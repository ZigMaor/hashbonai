# חשבונאי - הרפתקת חשבון ושפה! 🦊

אפליקציית לימוד אינטראקטיבית לילדים ישראלים, המשלבת חשבון ושפה עברית בממשק צבעוני ומהנה.

## מודולים

### חשבון 🔢
- **חיבור עד 100** - לומדים לחבר מספרים
- **חיסור עד 100** - לומדים לחסר מספרים
- **כפל בסיסי** - לוח הכפל 1 עד 5
- **בעיות מילוליות** - פותרים חידות חשבון
- **צורות גיאומטריות** - לומדים על צורות
- **שעון** - לומדים לקרוא שעון

### שפה 📚
- **קריאה** - קוראים ומבינים סיפורים
- **כתיב** - לומדים לכתוב נכון
- **דקדוק** - זכר ונקבה, יחיד ורבים
- **אוצר מילים** - מילים נרדפות, הפוכות, שורשים וביטויים
- **סיפור יצירתי** - כותבים סיפורים

## תכונות
- מערכת ניקוד ואבני דרך מעודדת
- תמיכה מלאה ב-RTL עברית
- התאמה מגדרית (בן/בת)
- הדרכה מדורגת לכל מודול
- מבחן סיכום בכל מודול
- כתיבה יצירתית עם גלריית סיפורים
- דוח התקדמות להורים
- נגישות מלאה (WCAG): ניגודיות גבוהה, גודל טקסט, הקראה, ניווט מקלדת

## טכנולוגיה
- HTML / CSS / JavaScript בלבד - ללא תלויות חיצוניות
- כל הנתונים נשמרים מקומית ב-localStorage
- אין שרת, אין cookies, אין tracking
- מאובטח עם Content Security Policy

## הפעלה
פשוט פתח את `index.html` בדפדפן, או העלה ל-GitHub Pages.
